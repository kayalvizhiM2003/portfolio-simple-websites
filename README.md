# Personal Portfolio Website

This is a simple personal portfolio website created using HTML and basic CSS. 
The website contains two main pages: Home and Contact.

## Home Page
The Home page introduces who I am and provides a brief overview of my portfolio.

## Contact Page
The Contact page allows visitors to contact me through a simple contact form.

## Technologies Used
- HTML
- CSS

## How to View
You can access the portfolio website by opening the **index.html** file in a web browser.
